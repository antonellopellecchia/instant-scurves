# Instructions

## Installation

```bash
git clone git@github.com:antonellopellecchia/instant-scurves.git
cd instant-scurve
source env.sh
poetry install
```

## Usage
```bash
cd instant-scurve
source env.sh
poetry shell
flask run --host="0.0.0.0"
```

Open http://localhost:5000 and follow the instructions  in the browser.
